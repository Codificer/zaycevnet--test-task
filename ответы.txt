1. Цвет текста hello World будет белым. Причина: у класса .MyBox выделен с помощью !important цвет белый, всамом html файле (.myBox {color: white !important;})
При этом, в css файле, по идее, есть более сильный по специфичности определитель цвета по ID элемента, с красным цветом текста - но у него вместо фигурных скобок - обычные круглые,
и поэтому он не применится. У остальных селекторов ниже вес.

2. на текущий момент, при наведении резко меняется длина текста, без учета времени transition. Причина - не указана изначальная ширина div.  
Достаточно прописать width:100% (ну или любую другую, конкретную) - и пример заработает. Системе просто нет стартовой величины, от которой требуется изменить плавно значение.
Поэтому, применяет сразу итоговый вариант.
